# Survey Insight Visualizer

A lightweight tool that turns messy, multilingual survey data into structured insights for faster product decisions.

## Why I built this
As a product designer, I often work with survey data across different languages.
However, translating, cleaning responses data, and visualize takes time and slows down decision-making.

This tool helps transform raw responses into clear patterns that teams can act on quickly.

## Features
- Upload CSV survey data
- Translate multilingual responses (ZH/JP → EN)
- Single question & Cross analysis
- Filter low-frequency responses
- Visualise insights:Chart & Table
- Download analysis result
- RWD UI
