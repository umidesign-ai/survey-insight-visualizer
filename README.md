# Survey Insight Visualizer

A lightweight tool that turns messy, multilingual survey data into structured insights for faster product decisions.

## Why I built this
As a product designer, I often work with survey data across different languages.
However, translating, cleaning responses data, and visualize takes time and slows down decision-making.

## Impact
This tool shortens the gap between raw survey data and decision-making, reducing analysis time from hours to minutes.
By replacing manual spreadsheet workflows and avoiding the need for additional analytics tools, it offers a cost-efficient solution for early-stage product exploration.
Its offline-first approach ensures that sensitive research data remains local, addressing privacy and security concerns when working with internal or user-generated data.

## Features
- Upload CSV survey data
- Translate multilingual responses (ZH/JP/EN/DE)
- Single question & Cross analysis
- Filter low-frequency responses
- Visualise insights:Chart & Table
- Download analysis result
- RWD UI
- Light/Dark mode
